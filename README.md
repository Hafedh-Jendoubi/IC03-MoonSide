# Moonside Connect

Intelligent Enterprise Social Network - Final Year Project

## 📋 Overview

Moonside Connect is an intelligent internal social network developed for Moonside Corporation. The platform aims to centralize company communication, facilitate knowledge sharing, and leverage AI to improve employee engagement and productivity.

## 🎓 Context

This is my end-of-studies graduation project (PFE) during my internship at Moonside Corporation. The project is being developed over 6 months with a focus on creating a modern collaborative platform for enterprise environments.

## 🔧 Tech Stack

### Backend
- **Java Spring Boot** - Microservices architecture
- **Python FastAPI** - AI services
- **Spring Data JPA** - Data persistence
- **Redis** - Caching

### Frontend
- **Next.js**
- **React**
- **TypeScript**

### Database & Messaging
- **MongoDB** - Main database
- **Kafka** - Event streaming
- **Elasticsearch** - Search engine
- **ClickHouse** - Analytics

## ✨ Core Features

- User authentication & profiles
- Posts, comments & social interactions  
- Teams & departments management
- Badges & gamification system
- Role-based permissions (RBAC)
- AI-powered content recommendations
- Advanced search capabilities
